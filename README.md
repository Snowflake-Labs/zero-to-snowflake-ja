# Zero to Snowflake - 日本語版

Snowflake の基本機能を学ぶためのハンズオンチュートリアル（日本語版）です。

## 概要

このリポジトリは、[Tasty Bytes - Zero to Snowflake](https://quickstarts.snowflake.com/guide/tasty_bytes_introduction/index.html) クイックスタートの日本語翻訳版です。架空のフードトラック会社「Tasty Bytes」のデータを使用して、Snowflake の主要機能を実践的に学習できます。

## コンテンツ

| ファイル | 内容 |
|---------|------|
| `01_tb_introduction.sql` | 環境セットアップ、データベース・ウェアハウス・ロールの作成 |
| `02_tb_show_resources.sql` | 各種リソースの確認コマンド |
| `03_tb_cost_management.sql` | コスト管理とリソースモニター |
| `04_tb_transformation.sql` | データ変換、クローン、タイムトラベル |
| `05_tb_governance_snowflake_horizon.sql` | Snowflake Horizon によるガバナンス（RBAC、マスキング、行アクセスポリシー） |
| `06_tb_semi_structured_data.sql` | 半構造化データ（JSON）の操作 |

## 前提条件

- Snowflake アカウント（トライアルアカウント可）
- ACCOUNTADMIN ロールへのアクセス権限

## 使用方法

1. Snowflake にログインし、Snowsight を開く
2. 各 SQL ファイルを順番に実行
3. コメントを読みながら、各ステップを理解する
